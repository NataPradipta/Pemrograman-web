Cara Mematikan Windows Defender di PC atau Laptop Kamu Windows 10

By Rajendra

Windows defender memang dibuat sebagai antivirus gratis dari windows untuk para penggunanya. Namun terkadang real-time protectionnya justru menghapus file-file dan mengurangi kenyamananmu. Jadi untuk kalian yang sedang mencari-cari cara menyelesaikan masalah ini, berikut adalah Cara Mematikan Windows Defender Windows 10 di PC atau Laptop Kamu.


Ada 3 cara untuk menuju menu windows defender pertama adalah dengan melakukan klik pada toolbar pojok kanan bawah, yang memiliki panah ke atas, lalu double klik gambar perisai milik windows.

Kedua kamu bisa menulis pada search dengan mengetik “Windows security”

Ketiga kamu bisa melalui start > Setting > Update Setting seperti gambar dibawah

Sekarang setelah mendapat menu seperti gambar dibawah kamu bisa double klik virus & Threat Protection di tengah atau ambil klik Virus & Threat Protection di kiri. Lebih gampangnya lihat saja gambar di bawah.

Setelah itu kamu akan bisa melihat gambar di bawah jika ingin mematikan tinggal di turn off, jika ingin diaktifkan tinggal turn on.

Ternyata tidak rumit bukan? Tapi ingat resiko untuk mematikan windows defender semua ada di kamu.

Jadi demikian Cara Mematikan Windows Defender Windows 10 di PC atau Laptop Kamu. Jadi apa kamu memiliki cara lain untuk mematikan windows defender atau mungkin kamu kubu yang tak pernah mematikan windows defender? Silahkan berikan komentar ya, semoga bermanfaat. Sekian dan Terima Kasih.
